# 🏠 Netherlands Housing Market Analysis

## 📌 Project Overview
Analyzed 6,800+ real Dutch housing records from Funda 
(the Netherlands largest real estate platform) to uncover 
price trends, patterns and built a price prediction model.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- SQL (SQLite)
- Plotly (Interactive Visualizations)
- Scikit-learn (Machine Learning)
- Google Colab (Development Environment)

## 📊 Key Insights
- 🏰 Villas are the most expensive house type (avg €1M+)
- ⚡ A+++ energy label homes cost 73% more than G label homes
- 📉 Average prices dropped from €539,464 (2022) to €457,670 (2023)
- 📐 Larger area generally correlates with higher price

## 📈 Visualizations
- Top 10 House Types by Average Price (Bar Chart)
- House Price Trend by Year (Line Chart)
- Average Price by Energy Label (Bar Chart)
- House Area vs Price (Scatter Plot)
- Actual vs Predicted Prices (ML Validation Chart)

## 🤖 Machine Learning Model
Built a Linear Regression model to predict house prices
based on area and number of rooms.
- R² Score: 0.21
- RMSE: €199,992
- Note: Model performance is limited due to absence of location-based features,
  which are critical in real estate pricing.

## 📂 Dataset
- Source: Funda - Sold Houses in Netherlands (Kaggle)
- Records: 6,823 cleaned records
- Original: 9,975 records
